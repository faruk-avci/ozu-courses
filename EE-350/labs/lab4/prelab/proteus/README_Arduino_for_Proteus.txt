To add the Arduino boards to Proteus, 
    1. extract the files inside the zip file
    2. copy them inside the LIBRARY folder under your Proteus installation

There will be two files:
    - ARDUINO.IDX
    - ARDUINO.LIB 

Example path for the LIBRARY folder:
C:\Program Files (x86)\Labcenter Electronics\Proteus 8 Professional\LIBRARY\
